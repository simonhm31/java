Java Notes

Getting Started:

Methods: functions that are part of a class

Classes: containers for one or more related methods

All Java functions are part of a class. Therefore, functions in Java are called "methods"

Methods in Java have the syntax:
AccessModifier ReturnType Name() {

}

Every Java program should have a main method
public void main(){

}

The main method is inside of class

public class Main {

public void main {

}
}

All methods and classes in Java should have an access modifier

Access Modifier: special keyword that determines if other classes and methods
in the program can access this method and class, (public, private)


Classes use: PascalNamingConvention

Methods use: camelNamingConvention

Types:

There are two kinds of types:

  • Primitive Types

    • Used for storing simple values

  • Reference Types:

    • Used for storing complex objects

Examples of Primitive Types:

Type Bytes Range
byte 1 [-128, 127]
short 2 [-32K, 32K]
int 4 [-2B, 2B]
long 8
float 4
